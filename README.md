# Laba 1 OS
### Задача
Написать консольную программу Main и две утилиты (служебные программы) Creator и Reporter, которые выполняют следующие действия.

#### 1. Утилита Creator
Создает бинарный файл, имя которого и количество записей в котором получает через командную строку. Записи файла вводятся с консоли и имеют структуру:

struct employee
{
    int num;        // идентификационный номер сотрудника
    char name[10];  // имя сотрудника
    double hours;   // количество отработанных часов
};

#### 2. Утилита Reporter
Создает текстовый файл, содержащий упорядоченный список сотрудников с начисленной заработной платой. Отчет формируется на основе бинарного файла, созданного программой Creator.

Аргументы командной строки:
1. Имя исходного бинарного файла.
2. Имя файла отчета.
3. Оплата за час работы.

Формат отчета:
- Заголовок: Отчет по файлу "имя исходного бинарного файла".
- Строки с данными: Номер сотрудника, имя, часы, зарплата.

#### 3. Программа Main
Координирует работу утилит:
1. Запрашивает имя бинарного файла и количество записей.
2. Запускает Creator (через CreateProcess), передавая параметры в командной строке.
3. Ждет завершения (WaitForSingleObject).
4. Выводит содержимое бинарного файла на консоль.
5. Запрашивает имя файла отчета и ставку оплаты.
6. Запускает Reporter, передавая параметры.
7. Ждет завершения.
8. Выводит отчет на консоль.
9. Завершает работу.

## Технический стек

- Язык: C++ (Standard: C++17)
- Платформа: Windows (Windows API)
- Система сборки: CMake

## Структура проекта

Решение состоит из трех проектов:
```text
Project Root
├── apps/
│   ├── main/       # Main: Основной управляющий процесс
│   ├── creator/    # Creator: Утилита для создания бинарных файлов данных
│   └── reporter/   # Reporter: Утилита для чтения данных и генерации отчетов
└── libs/
    ├── employee.h  # Структура описания сотрудника
    ├── fileUtils.h # Функции для работы с файлами
    └── process.h   # Класс-обертка для управления процессами
```
